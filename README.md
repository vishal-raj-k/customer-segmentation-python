# Customer Segmentation using K-Means Clustering

## Project Overview

This project demonstrates customer segmentation using machine learning.
Customers are grouped based on **Income** and **Spending Score** to identify behavioral segments.

## Tools Used

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Method

The clustering algorithm used is **K-Means**, which groups customers into clusters based on similarity.

## Output Visualization

![Customer Segmentation](customer_segments.png)

## Key Insights

* Customers are divided into multiple spending groups.
* High-spending and low-spending segments can be identified.
* Businesses can use these insights for targeted marketing.
